# Learn Chinese

An interactive web application for kids (ages 8-12) to learn Traditional Chinese through vocabulary, sentences, stories, and quizzes.

## Features

- **60 Comprehensive Lessons** - Organized into 3 difficulty levels across 20 themes
- **Interactive Vocabulary** - Learn characters with Jyutping (Cantonese) and Pinyin (Mandarin) romanization
- **Sentence Practice** - Word-by-word breakdowns with translations
- **Short Stories** - Engaging narratives reinforcing lesson vocabulary
- **Quizzes** - Multiple choice and matching questions to test knowledge
- **Progress Tracking** - Track completion across all lessons
- **Dark Theme** - Easy on the eyes for comfortable learning

## Curriculum Structure

### Part 1: Basic Level (Lessons 1-20)
Simple vocabulary and sentences for beginners.

### Part 2: Intermediate Level (Lessons 21-40)
More complex vocabulary and sentence structures.

### Part 3: Advanced Level (Lessons 41-60)
Sophisticated language and advanced concepts.

### Themes Covered

| Category | Themes |
|----------|--------|
| Basic Environment | At Home, My Family, At School, My Friends, Playground |
| Daily Activities | Times of Day, Mealtime, Shopping, My Body, Getting Dressed |
| Food & Culture | Restaurant, Dim Sum, Fruits & Vegetables, Festivals, Cooking |
| Activities & Hobbies | Sports, Animals, Transportation, Weather, Hobbies |

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development and building
- **Tailwind CSS** for styling
- **Zustand** for state management with localStorage persistence
- **Framer Motion** for animations
- **React Router v6** for navigation
- **to-jyutping** and **pinyin** for romanization

## Development

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
